## 类型
- 在C++中，每个名字都有一个与之相关联的类型，类型决定操作

### 基本类型
- 布尔、字符、整数、浮点、枚举、void、指针、数组、引用
- interger type: 布尔量、字符、整数类型
- 算术类型：整型 + 浮点
- 用户定义类型：浮点和类
